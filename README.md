# 🏦 Bank Management System (Java)

A console-based Bank Management System built using Java and Object-Oriented Programming concepts.

---

## 🚀 Features

* Create Account
* Deposit Money
* Withdraw Money
* Display All Accounts

---

## 🧠 Concepts Used

* Classes & Objects
* Constructors
* Inheritance
* Method Overriding
* Arrays of Objects
* Conditional Logic & Loops

---

## ▶️ How to Run

1. Compile all files:

```
javac src/*.java
```

2. Run the program:

```
java -cp src Main
```

---

## 📌 Example Usage

```
1 → Create Account
2 → Deposit
3 → Withdraw
4 → Display All
5 → Exit
```
